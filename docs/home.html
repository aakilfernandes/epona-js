Phapplejax

is a PHP+Bootstrap+AJAX boilerplate for single page web apps. I wrote Phapplejax for karmanetics in the hopes I'd never have to write another sign up form.

Features

Security

  • Iterated Sha256 password hashing
  • Random salting for each users
  • mysqli database connections

Single Page

  • Modal-driven and mobile friendly
  • URL hash route management (via epona.js)
  • AJAX form submissions (via jqueryForm.js)
  • Easy to integrate with Backbone.js

User Account Management

  • Enforces single-username, single-password policy
  • Transactional emails via Mandrill (free with limits)
  • Generates email confirmations and password reset links
  • SQL for database set up
